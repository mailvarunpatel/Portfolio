# Varun Patel — Portfolio Website

Personal portfolio website for [countrep.com](https://countrep.com).

## About

A single-page portfolio showcasing my product management career spanning 14+ years in IoT, connectivity platforms, satellite, and enterprise SaaS across Africa, Asia, and Europe.

## Tech Stack

- Pure HTML, CSS, and JavaScript (no build tools required)
- Google Fonts (Playfair Display + Source Sans 3)
- Hosted on Netlify with custom domain

## Local Development

Simply open `index.html` in your browser. No server or build step needed.

## Deployment

This site auto-deploys to Netlify on every push to the `main` branch.
